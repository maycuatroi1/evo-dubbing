evo-dubbing

AI voice-over dubbing for online videos, starting with YouTube. Reads captions, translates, and speaks the translation over the original using your own OpenAI or Gemini API key. This page always serves the latest build from main.

Download extension (.zip)

Install

  1. Download and unzip the package above.
  2. Open chrome://extensions and turn on Developer mode.
  3. Click "Load unpacked" and select the unzipped folder.
  4. Open the extension options and paste your OpenAI or Gemini API key.
  5. Open a YouTube video and use the floating panel to dub it.

Notes

  • API keys stay in your browser and are never sent to any server.
  • Works best on videos that have captions (most YouTube videos do).
  • You can optionally share a finished dub so others can reuse it.
